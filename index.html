Introduction

Welcome to the guidance document for our project. This page is hosted for free using GitHub Pages—no domain purchase required!

Setup Steps

  1. Create a GitHub repo named username.github.io (replace username with your GitHub handle).
  2. Clone the repo locally or upload files via the GitHub web interface.
  3. Add this index.html into the root of the repo.
  4. Commit and push your changes.
  5. In the repo’s **Settings → Pages**, ensure **Branch: main** (or **master**) and **/ (root)** are selected, then save.
  6. Visit https://username.github.io/ in your browser—your page is live!
