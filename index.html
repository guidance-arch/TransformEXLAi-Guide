Setup & Initial Walkthrough

  1. File Uploads

    You can upload files in the following formats: .xlsx, .xls, .csv.

    The maximum file size is currently 50 MB per file. For larger datasets, consider splitting files or uploading sequentially.

    Tip: If you need to process larger files, split them into smaller ones or ensure a stable network during uploads.

  2. Processing Time

    Processing time depends on file size and complexity:

    • Under 5 MB: Approximately 30 seconds to 1 minute.
    • Up to 50 MB: May take a few minutes.

    This includes intelligent data extraction, table detection, and schema inference powered by advanced AI algorithms.

  3. Setting Up Workspaces

    Workspaces are your personal areas to explore and query data:

    • Create a workspace by uploading your spreadsheet and naming the workspace.
    • The system auto-detects tables and presents them as separate entities.
    • Work with tables individually or collectively based on your preference.

    Currently, each user has access to one workspace; additional workspaces are planned based on feedback.

  4. Semi-Auto & Auto Modes

    Choose your workflow mode:

    • Auto Mode: Queries run across all detected tables simultaneously.
    • Semi-Auto Mode: Select specific tables and columns to target your queries.

  5. Plotting & Saving Results

    After executing a query (generated or manual SQL), view results in the results pane. Suggested plot types include pie charts, bar graphs, scatter plots, and more. You can save result sets and visualizations for future reference.

  6. Supported Data Structures

    Supported formats include:

    • Single table per sheet.
    • Multiple sheets in one file.
    • Multiple tables in a single sheet (separated by empty rows or columns).

    Best Practice: Ensure clean layouts and consistent header rows for optimal detection.

  7. Unsupported Content

    Avoid uploading files containing:

    • Embedded images or charts (these are ignored).
    • Complex macros (these will not execute).
    • Password-protected files (remove protection first).
